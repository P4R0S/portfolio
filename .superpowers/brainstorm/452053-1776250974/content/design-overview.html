// animation system
welcome text
Types out character-by-character on first load. ~40ms per char. Skippable with Enter.
command input
User types freely. Blinking amber block cursor. Tab autocompletes command and filename.
command output
Output types line-by-line at 20ms/char. Long outputs stream faster. Skippable.
↑ / ↓ arrows
Command history navigation, exactly like a real shell.
CRT effect
Subtle scanline overlay + very slight vignette. Optional: user can type `crt off`.
prefers-reduced-motion
All typewriter effects disabled. Output appears instantly. Cursor still blinks.
